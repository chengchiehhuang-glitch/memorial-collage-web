# 回憶拼貼 memorial-collage-web

把一批照片做成「中心人物 + 四周環繞」的復古剪貼簿紀念/感恩拼貼海報或影片首頁圖。
**100% 純前端、照片不上傳**（全在瀏覽器處理），可離線使用。

## 用法
直接開 `index.html`（或起個靜態 server）：
```
python3 -m http.server 4761 --directory .
# 開 http://localhost:4761
```
上傳照片 → 自動排「中心＋四周」→ 拖拉/滾輪縮放/Shift+滾輪旋轉微調 → 匯出 PNG。

## 功能（v2）
- 多張上傳、拖放；點照片列設「中心人物」
- 版面比例：16:9 / A4 / 正方形 / 4:3
- 內建復古紙底圖（assets/bg）＋可上傳自訂底圖
- 標題／副標／落款文字（LXGW 毛筆字，可拖到任意位置）
- 高解析 PNG 匯出（canvas 2×，含白框拍立得＋陰影）
- 行動裝置觸控拖曳

## 技術
- 單頁 vanilla JS，無後端、無框架、無追蹤。
- 字體 LXGW 霞鶩文楷 TC（OFL）。底圖為 AI 生成（無人臉）。
